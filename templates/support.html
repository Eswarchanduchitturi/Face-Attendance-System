About Face Attendance System

How the Application Works
  • The system captures live video through the camera during check-in or check-out sessions.
  • Each frame is processed using Haar Cascade to detect face regions.
  • Detected faces are converted into grayscale and preprocessed for better recognition.
  • Facial features are extracted using LBPH (Local Binary Pattern Histogram).
  • Extracted features are compared with trained employee facial data from the model.
  • If identity is verified, attendance is marked as Check-In or Check-Out.
  • The system calculates worked hours from login and logout timestamps.
  • Records are stored securely with date, day, time and images.
Core Technologies Used
  • Python + Flask
  • OpenCV + Haar Cascade
  • LBPH Face Recognizer
  • SQLite Database
  • Bootstrap + Custom UI
System Features
  • Smart Check-In / Check-Out mode
  • Auto worked-hours calculation
  • Live face verification
  • Recognition monitoring and analytics
  • Attendance history and PDF export
  • Enrollment and model training
  • Complaint and feedback management
