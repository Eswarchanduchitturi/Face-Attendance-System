About Face Attendance System

How the Application Works
  • The system captures live video through the camera during check-in or check-out sessions.
  • Each frame is processed using the Haar Cascade algorithm to detect facial regions.
  • Detected faces are converted into grayscale and preprocessed for better recognition accuracy.
  • Facial features are extracted using the LBPH (Local Binary Pattern Histogram) algorithm.
  • Extracted features are compared with trained employee facial data stored in the model.
  • If the identity is verified, attendance is marked either as Check-In or Check-Out.
  • The system automatically calculates worked hours based on login and logout timestamps.
  • All records are securely stored in the database with date, day, and time details.
Core Technologies Used
  • Python & Flask (Backend Framework)
  • OpenCV (Computer Vision Processing)
  • Haar Cascade (Face Detection)
  • LBPH Face Recognizer (Feature Extraction)
  • SQLite Database (Data Storage)
  • Bootstrap & Custom CSS (Modern UI)
